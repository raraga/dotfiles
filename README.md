## Installation Notes

### While using NeoVim
* Symlink `~/.vimrc` to Neovim configuration

    ln -s ~/.vimrc ~/.config/nvim/init.vim

### Querks with Vim Themes
* `colorscheme` is very finnicky unless you create a `colors` directory first with copies of the selected theme.
